# Word 输出格式规范

> 适用于外文文献编译报告的 .docx 输出 | 可复用于其他公文场景

## 一、页面设置

| 属性 | 值 |
|------|-----|
| 纸张 | A4（21cm × 29.7cm） |
| 上边距 | 2.54 cm |
| 下边距 | 2.54 cm |
| 左边距 | 3.17 cm |
| 右边距 | 3.17 cm |

## 二、字体

| 元素 | 中文字体 | 英文字体 | 字号 | 加粗 |
|------|---------|---------|------|------|
| 报告标题 | 方正小标宋简体 | Times New Roman | 二号（22pt） | 否 |
| 一级标题（一、二…） | 黑体 | Times New Roman | 三号（16pt） | 否 |
| 二级标题（（一）（二）…） | 楷体_GB2312 | Times New Roman | 三号（16pt） | 否 |
| 三级标题（1. 2. …） | 仿宋_GB2312 | Times New Roman | 三号（16pt） | 是 |
| 正文 | 仿宋_GB2312 | Times New Roman | 三号（16pt） | 否 |
| 摘要/关键词标签 | 仿宋_GB2312 | Times New Roman | 三号（16pt） | 是 |
| 引用行 | 楷体_GB2312 | Times New Roman | 小四（12pt） | 否 |
| 标签（附录） | 仿宋_GB2312 | Times New Roman | 小五（9pt） | 否 |

## 三、段落

| 属性 | 值 |
|------|-----|
| 对齐方式 | 两端对齐 |
| 首行缩进 | 2 字符（三号仿宋 ≈ 32pt ≈ 1.13 cm） |
| 行间距 | 固定值 28 磅 |
| 段前间距 | 0 行 |
| 段后间距 | 0 行 |

## 四、标题格式

| 级别 | 编号格式 | 示例 | 段落格式 |
|------|---------|------|----------|
| 报告标题 | 无编号，居中 | 关于XX《XX》的分析 | 居中，上下各空一行 |
| 一级标题 | 中文数字 + 、 | 一、引言与背景 | 首行缩进 2 字符 |
| 二级标题 | 括号中文数字 | （一）总体规模 | 首行缩进 2 字符 |
| 三级标题 | 阿拉伯数字 + . | 1. 数据来源 | 首行缩进 2 字符 |

## 五、特殊元素

| 元素 | 格式 |
|------|------|
| 摘要正文 | 首行缩进 2 字符，与"摘要："标签同行 |
| 关键词 | 首行缩进 2 字符，"关键词："加粗，内容不加粗 |
| 引用行（标题下） | 居中，灰色，楷体小四 |
| 正文中的数字/英文 | 使用 Times New Roman，半角 |
| 表格 | 表头加粗居中，正文左对齐，单倍行距，宋体五号 |

## 六、命名规则

| 场景 | 格式 |
|------|------|
| 输出文件名 | `{YYYY-MM-DD} {来源}-{报告简称}.docx` |
| 示例 | `2025-12 EBA-ECB-Payment-Fraud-Report.docx` |
